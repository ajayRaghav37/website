How to modify source code?

After downloading the source files (as .zip compressed file), extract it and open "READ ME.txt". The instructions for modifying the source of that particular program are available in that file. Most of the programs are developed in Visual Basic 98 (aka VB6) that can be downloaded and learned from YouTube. You can modify the source code and redistribute it as long the terms of GPL are not violated. Once you modify the source, you can choose from:

  1. Contact Us (If the modification is minor)
  2. Send us the modified file here (If the modification is major)
  3. Keep it (If you modified a Private Software)
  4. Redistribute a new copy (Strictly as Open Source Software)

You need not worry about the credits; we will automatically add your name in the about dialog of the program or on the site's About Us page (only if your modification is found useful). So, let us take the maximum advantage of Open Source Software and develop the software we use!

Direct Download via TinyURL

Most of the content of ANIco.in can be downloaded directly by short URLs provided by TinyURL.com. The logical steps for downloading the latest version of a particular file without opening the official website is as follows:

  1. Open the web browser and type www.tinyurl.com in the address bar
  2. Type a slash and then type anico
  3. Type a dash and then type the official abbreviation of the product you want to download
  4. Pressing enter will start the downloading of that particular program's latest version
  5. If you want some other file related to that program, type a dash and then type the tag for that file and press enter to start downloading that file. (Tags: Source (s), Older Version (o), etc.)
